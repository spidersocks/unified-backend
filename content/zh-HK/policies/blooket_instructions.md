# Blooket（網上遊戲）功課說明

**重要：如家長或學生查詢 Blooket 或網上遊戲，請務必在答案最後另起一行加上此標記：**
```
[SEND_BLOOKET_PDF]
```

例子：  
問：點樣玩網上遊戲？  
答：請參考我們的 Blooket 教學指引。  
```
[SEND_BLOOKET_PDF]
```

完成功課後，請點擊「End Now」退出。  
在截止日期前可重複練習。
