# 班級人數（小班FAQ）

問：每班有幾多個學生？  
答：我們所有課程均採用小班教學。一般每班4至6人（部分高年級課堂最多8人）。

問：可否安排私人／一對一課堂？  
答：可以。如有需要可聯絡我們查詢。

問：全部課程都係小班教學嗎？  
答：是。我們重視每位學生的學習體驗，因此所有課堂均以小班授課。

問：最多有幾多個學生？  
答：一般為4至6人，如有需要或高年級課程可最多8人。

問：一堂課多長時間？一星期幾堂？
答：課程的時長從45分鐘到1個小時不等。具體課程會根據學生年齡和水準決定。
