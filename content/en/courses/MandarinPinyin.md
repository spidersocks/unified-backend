# Mandarin Pinyin (魔法拼音班)

### Overview
Intensive course on Hanyu Pinyin (Romanization system) for Mandarin Chinese, essential for pronunciation and input methods.

### Target age
4 years old / up (typically K2 and above)

### Class size
Small group class size, 4–6 students per class

### Tuition
$3,200 / 8×45m; $4,500 / 10×1h

### Integration with regular Chinese pathway
For Mandarin learners, Pinyin is normally introduced within the regular Chinese (language) pathway around K2 (readiness‑based) and consolidated in K3. This standalone Pinyin class provides focused reinforcement for tones, initials/finals and accurate pronunciation.
