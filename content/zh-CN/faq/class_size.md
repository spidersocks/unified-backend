# 班级人数（小班FAQ）

问：每班有多少个学生？  
答：我们所有课程均为小班教学。每班一般4到6人（部分高年级课程最多8人）。

问：可以安排私人／一对一课程吗？  
答：可以。如有需要请联系我们。

问：所有课程都是小班吗？  
答：是的。我们注重每位学生的学习体验，全部课程均为小班授课。

问：最多多少学生？  
答：通常每班4到6人，少数高年级课程可最多8人。
